Magento admin action system

PPL Action Widget

Action Widget creates Magento admin cards from shop data. Each card can show severity, score, evidence, recommendation and a link to the related admin area.

Version 1.0.0 PPL Hub PPL Report Core Custom software license

From data to action

The module uses detectors. Each detector checks one topic, stores a snapshot and creates cards in the admin.

Admin path: PPL Hub > PPL Action Widget > Action Widgets.
  • Action cards: severity, score, impact, evidence, recommendation and CTA per finding.
  • Snapshot calculation: cron-based builds avoid heavy live admin queries.
  • Detectors: zero-result searches, viewed-not-bought products, abandoned carts, stock risk, content gaps and more.
  • Admin controls: threshold configuration, per-admin dismiss actions and CSV export.
