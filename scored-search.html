Magento search extension

PPL Scored Search

A Magento search module with weighted attributes, MySQL and OpenSearch support, search modes and an AJAX result overlay.

View on marketplace Request support
Version 1.1.1 Magento 2.4+ PHP 8.1+ Custom software license

What it does

PPL Scored Search replaces plain default matching with configurable ranking logic. Store admins can set attribute weights, compare search modes and inspect scoring details in the admin.

Search engines

Supports MySQL and OpenSearch with automatic or manual engine selection and fallback logic.

Search modes

Default, Simple and Smart modes allow different ranking strategies depending on store setup and performance goals.

Weighted ranking

Attribute weights define how product fields influence the final result order.

Frontend overlay

AJAX live search, mini product detail view, variant rendering, wishlist, compare and add-to-cart support.

Key capabilities

Search settings and diagnostics are available in the Magento admin.

Test Lab

Compare behavior across search modes and engines and inspect scoring details.

Variant handling

Color and size options can be displayed in the overlay with validation for add-to-cart flows.

Theme controls

Modular CSS supports layout, theme, density and radius settings without forcing a single storefront style.
