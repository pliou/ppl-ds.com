Module map

A quick overview of how the release modules fit together.

Search ownership

Search Intelligence controls query understanding, result basis, zero-result recovery, redirects, merchandising and diagnostics. Discovery Filters handles indexed filtering and facet presentation.

Consent-aware signals

Tracking & Privacy can collect first-party behavioral events inside Magento when enabled and consented, then aggregate them for privacy-aware operational use.

Operational action board

Action Widget and Dashboard Widgets Pro surface prioritized cards, outside operational signals and board health information from PPL report sources.

Reports stack

Report Core provides the shared report shell, registry, period controls, export behavior, charts and scheduled email infrastructure for report modules.

Order Metrics

Order, catalog, customer, review, cancellation, sales-rule and performance diagnostics in one admin analytics surface with scheduled CSV delivery.

Dev setup

Quick Setup orchestrates generator modules into resumable scenarios for QA, demo and disposable staging shops.
