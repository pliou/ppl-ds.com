Magento development and QA utilities

PPL Developer Tools

Magento admin tools for local development, QA, demo stores and disposable staging systems. Production mode is blocked unless it is explicitly allowed.

Included modules

The tools are split into a shared shell and separate data-generation modules.

Dev Tool

Shared admin hub, toolbar styling, navigation, search-term simulator and generated-data cleanup.

Dev Create Orders

Create seed orders for reports, search tests and admin checks.

Dev Create Customers

Create seed customer records for QA, reports and order checks.

Dev Create Articles

Create seed products and article-like catalog entries for storefront and search testing.

Dev Create Store

Prepare development store structures and demo setup data.

Safety model

Generated-data cleanup defaults to dry-run and production execution is blocked unless explicitly allowed.

Generate, test and clean up

The tools support generated test data, admin checks and cleanup in controlled Magento environments.

Create Orders

The Create Orders module generates seed order data for QA, report validation, demo flows and staging checks in controlled Magento environments.

Create Customers

The Create Customers module generates seed customer records for local development and QA, including profile data, review steps and records that can be cleaned up again in non-production stores.

Create Articles

The Create Articles module builds repeatable seed catalog entries with titles, descriptions, placeholder images, prices, stock status and categories for non-production Magento environments.

Create Store

The Create Store module prepares repeatable development store setup data, including store settings, catalog options and validation steps for non-production systems.
