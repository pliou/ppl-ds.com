Magento checkout extension

PPL Order Comment

A checkout module that adds an order comment field to Magento and stores the customer's note with the order.

Version 1.0.0 Checkout Admin configuration Custom software license

Checkout field

Adds a customer-facing order comment input during checkout.

Order storage

Preserves the submitted comment with the order data for later admin use.

Configuration

Provides admin configuration under PPL Hub for module settings.

Checkout notes saved with the order

The module adds a comment field during checkout, saves the note with the order and makes it available in the Magento admin order view.
