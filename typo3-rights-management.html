What it manages

For TYPO3 installations where selected backend users manage users, groups, modules and mounts.

Backend users

Review and adjust backend user rights and group assignments.

Backend groups

Manage group inheritance, module access and record permissions.

Mounts

Handle DB mounts, file mounts and page tree access in a structured interface.

Delegated writes

Non-admin users can be allowed to save rights only within effective permission boundaries.

History

Successful save operations are written to a history table with before and after payloads.

TYPO3 packaging

Composer package ppl/ppl_rights_management with extension key ppl_rights_management.
